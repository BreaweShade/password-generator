# Генератор паролей с оценкой сложности на основе ML

Десктопное приложение на Python с графическим интерфейсом для генерации безопасных паролей и их оценки с помощью машинного обучения (Random Forest).

## Установка

pip install -r requirements.txt

## Запуск

python password_generator.py

## Тестирование

python -m unittest test_password_generator.py

## Особенности
Генерация с настраиваемыми параметрами (длина, типы символов)
Оценка надёжности на основе ML-модели (Random Forest, 9 признаков)
Сохранение паролей в зашифрованном виде (Fernet)
Графический интерфейс на Tkinter
Автоматические тесты (unittest)
